# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

צוות סוכנים ליצירת תוכן: סוכן ראשי (מנכ"ל) שמנהל צוות סוכנים שמייצרים תוכן יחד. פרטי הסוכנים, התפקידים והזרימה ביניהם יתווספו ל-CLAUDE.md בהדרגה ככל שהפרויקט מתפתח.

## ⚠️ חובה: שימוש ב-`obsidian-vault-workflow`

הפרויקט מתועד כ-Obsidian vault תחת `vault/`. בכל סשן ובכל פקודה חדשה — **הפעל את הסקיל `obsidian-vault-workflow`** לפני שמתחילים לעבוד:

1. **לפני הפעולה** — קרא את `vault/Meeting Notes/_index.md` ואת קובץ ה-topic הרלוונטי לבקשה. במידת הצורך גם 2–3 ה-Meeting Notes האחרונים.
2. **אחרי הפעולה** — הוסף `### YYYY-MM-DD — <כותרת> [status]` ל-`## Session Log` של ה-topic file המתאים, עדכן `## Open Questions`, ושמור על `[[wikilinks]]` עדכניים.
3. **קובץ חדש בפרויקט** → צור MD מתאים תחת `vault/Meeting Notes/` (או תיקייה רלוונטית) **באותו commit**, והוסף שורה ל-`_index.md` ול-`vault/Meeting Notes/project-file-map.md`.

הסקיל המלא: `.claude/skills/obsidian-vault-workflow/SKILL.md`. כניסה ל-vault: [[vault/Meeting Notes/_index.md]].

## `.claude/` directory

הגדרות מותאמות-פרויקט ל-Claude Code יושבות תחת `.claude/`:

- `.claude/agents/` — הגדרות subagents ייעודיים לפרויקט (כרגע ריק).
- `.claude/skills/` — skills מותאמים. כרגע: 14 סקילים מ-`obra/superpowers` + `obsidian-vault-workflow` פנימי.
- `.claude/commands/` — slash commands מותאמים (כרגע ריק).
- `.claude/settings.local.json` — הרשאות מקומיות (autogenerated, לא בריפו).

תיעוד מפורט: [[vault/Meeting Notes/claude-config-structure.md]] ו-[[vault/Meeting Notes/superpowers-skills.md]].

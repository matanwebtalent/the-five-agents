# Claude Config Structure

## Overview

תיקיית `.claude/` בשורש הפרויקט מכילה את כל ההתאמות הספציפיות-לפרויקט של Claude Code:

- `.claude/agents/` — הגדרות subagents מותאמים (כרגע ריק, רק `.gitkeep`).
- `.claude/commands/` — slash commands מותאמים (כרגע ריק, רק `.gitkeep`).
- `.claude/skills/` — סקילים מותאמים. כרגע מכיל **15 סקילים**: 14 שהועתקו מ-`obra/superpowers` + סקיל אחד פנימי (`obsidian-vault-workflow`).
- `.claude/settings.local.json` — קובץ הרשאות מקומי של Claude Code (autogenerated). **לא בריפו**, ב-`.gitignore`.

הקבצים תחת `.claude/` נטענים אוטומטית על ידי Claude Code כשפותחים סשן בשורש הפרויקט.
